# LayaMiniGame

#### 介绍

    Laya小游戏框架
    简化了LayaBox的工作流程，让各个工具使用更加方便。

#### 参与贡献

- 辥觻瓭: https://gitee.com/lxqAxld_admin
- YaYaLuoYa: https://gitee.com/yayaluoya

#### 主要功能
    - 简化LayaBox开发流程
    - 在untiy中搭建场景并导出场景配置文件，根据配置文件自动构建场景，
    - 导出unti预制体文件
    - 根据untiy预制体场景导出预制体名字列表TS文件
    - 根据Excel文件导出配置表JOSN文件，并在游戏初始化时加载
    - 导出FGUI页面并生成TS文件
- 详情移步
